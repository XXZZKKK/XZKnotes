    Note 1

    显微镜中所观察的一个分子，一般数学上被定义为点光源，用狄拉克函数表示，显微镜包含多个透镜与孔隙，点光源发出的光线会发生像差、衍射、折射等行为，从而最终观察到的是一个模糊的光斑，而点扩散函数就是用来描述一个“完美的点”在经过光学系统后，是如何“扩散”成一个“模糊的斑”的数学函数，系统的PSF越集中，成像越清晰；PSF越弥散，成像越模糊。

    \[PSF = 成像系统对一个点输入的响应\]

    理论解析

    任何复杂场景，都可以看作是由无数个强度不同的“点光源”组成的集合，同时光学系统通常被近似为线性非时变系统（可线性叠加，满足空间平移不变，以及输入一定延迟，输出作相同延迟，波形不变）。传感器捕获到的二维图像 \(I(x,y)\)，实际上是理想的三维场景在二维投影上的真实分布 \(O(x,y)\) 与该系统的点扩散函数 \(\text{PSF}(x,y)\) 的卷积，并叠加了系统噪声 \(N(x,y)\)：

    \[I(x,y) = O(x,y) * \text{PSF}(x,y) + N(x,y)\]

    所以如果能精确推导系统的PSF，就能通过去卷积算法，从观测图像中逆向反推，重建精确的原始场景\(O(x,y)\)

    什么，为什么是卷积呢？重新给我自己回顾一下卷积的意义。

    \[y(t)=\int_{-\infty}^{+\infty}x(τ)h(t−\tau)d\tau\]

    \(t\)：我想看的输出的位置 \(\tau\)：输入x位置一个个扫过去的遍历变量 \(x(\tau)\)：输入在\(\tau\)位置的强度 \(h(t-\tau)\)：\(\tau\)位置，对输出位置\(t\)所产生的响应

    对于某个\(t\)位置上的响应，等于所有位置上的响应乘以这些位置相对于\(t\)的响应后求和，换句话说就是加权求和。我好像写着写着终于理解了

    相当于物体上每一个亮点，都会在图像上盖一个PSF模板的印章，重复在这张纸上把所有点的章全盖一遍，最终会得到所有模板的叠加，即为最终图像\(I(x,y)\)

    从信号处理视角来看

    卷积核是空间域滤波器，其傅里叶变换是OTF(optical transfer function)，OTF模长是MTF(modulation transfer function)

    \[\text{OTF}(f_x, f_y) = \mathcal{F}\{\text{PSF}(x,y)\}=MTF(f_x, f_y)\cdot e^{jPTF(f_x, f_y)}\]
    \[\text{MTF}(f_x, f_y) = |\mathcal{F}\{\text{PSF}(x,y)\}|\]
    \[\mathcal{F}\{I\}=\mathcal{F}\{O\}\mathcal{F}\{PSF\}\]

    如果PSF很弥散，高频细节的点发出的光会相互混合，导致信息丢失。理论而言，空间域和频域之间存在一个缩放特性，如果一个函数在空间域被拉宽（扩展），那么它的傅里叶变换在频域中必然被压缩（收缩）：

    \[\mathcal{F}\{f(a x)\} = \frac{1}{|a|} F\left(\frac{f_x}{a}\right)\]

    MTF下降，即对比度的下降，则会导致最终图像无法分辨高频细节。

    高频细节：高频图像中亮度剧烈、快速变化的区域。比如锐利的边缘、细密的毛发、粗糙材料表面的微小纹理
